# Logan's Portfolio

This is an awesome portfolio website created with Next.js and Shadcn

## Getting Started

Install dependencies with `npm install` and run the development server with `npm run dev`:

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Deployment

Deployment is automatically handled and 
the website will be automatically compiled and deployed
with a commit to the main branch.
